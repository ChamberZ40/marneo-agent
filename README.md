# Marneo Agent

> **Mare**（马）+ **Neo**（新）= **新马** 🐴

Marneo 是**项目数字员工**，不是个人助理。你招聘 TA、分配项目，TA 为项目工作、汇报、成长。

## 快速开始

```bash
pip install -e .
marneo setup      # 配置 LLM Provider
marneo work       # 与数字员工对话
```

## 命令

```
marneo setup          配置 Provider（交互向导）
marneo work           与员工对话
marneo --version      查看版本
```

## 路线图

- **Phase 1** ✅ — 基建：CLI、Provider 配置、对话 TUI
- **Phase 2** — 员工系统：hire 面试、四级成长、日报
- **Phase 3** — 项目系统：projects new、并行多项目、Skill
- **Phase 4** — Gateway：Feishu、WeChat、Telegram、Discord
- **Phase 5** — 打磨与生态
